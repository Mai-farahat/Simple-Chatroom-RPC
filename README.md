# 🗨️ Simple Chatroom using Go RPC

A simple chatroom application built in Go using Remote Procedure Calls (RPC).  
Clients can send and fetch chat messages from a shared server.

---

## 🚀 How to Run

### Server & Client
```bash
go run server.go
go run client.go

```
